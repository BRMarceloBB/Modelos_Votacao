Projeto Moda Sustentável: Sobre o Projeto


Este projeto faz parte do Segundo Festival de Audiovisial Estudantil de Vespasiano! Nós da Escola Municipal Bárbara Maria Salomão fizemos um projeto de Moda Sustentável com um desfile de modelos.

Este site é dedicado a mostrar como foi feito o projeto e à votação para sabermos qual modelo foi a melhor!
